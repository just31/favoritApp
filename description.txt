Приложение с выводом блочков по учебным предметам. И просмотром выбранного курса.

Используемые технологии:
angular.js,
bootsrap3.

Создал верстку идентичную указанной странице https://www.imumk.ru/showcase. Верстка резиновая.

Данные в блочки с описанием курсов, подгружаются асинхронно на angular.js с помощью сервиса '$http.post'.
С помощью директивы 'ng-repeat', рендеряться во view. Есть разделение по шаблонам, через '$routeProvider'.
Данные из шаблонов, используя маршрутизацию грузятся в '<div ng-view></div>'.

Для общего вывода, к блочкам не хватало изображений.
Чтобы реализовать его, создал директиву angular <load-images></load-images>.
Скачал несколько url-картинок со страницы https://www.imumk.ru/showcase, соответствующих предметам в выпадающем списке "Все предметы".
По классам распределение по картинкам не делал, только по предметам. Чтобы разнообразить внешний вид блочков с предметами.
Все url добавил в js-массив, откуда они выводятся по соответствию с предметами, к блочкам, через директиву <load-images></load-images>.

Фильтрация и поиск курсов происходят на лету, без перезагрузки страницы. Используя встроенные директивы angular и связи между моделями. 
Данные для <option value=""> списков, также берутся из модели контроллера. 
Используя встроенную директиву 'ng-options', добавляются в селекты.
Для поля поиска, убрал кнопочку с лупой. Т.к. в данном случае, поиск производится на лету, набирая в инпут необходимое слово или словосочетание.

На страницу добавлен, переключатель оплаты рубли/бонусы. В виде 2-х стилизованных чекбоксов.
Если выбрано "в рублях", второй переключатель "в бонусах", не активен.
И наоборот, если выбрано "в бонусах", неактивен переключатель в рублях. Это сделано, чтобы можно было вывести только одно значение в блочек с синей кнопкой.
Также для данной кнопки предусмотрено, значение по умолчанию "Купить за...".

Также немного попробовал усложнить задание и добавил дополнительно, просмотр выбранного курса. Переход к просмотру происходит от ссылок "Попробовать" и "Подробнее".
Функционалность также реализована на angular, используется сервис маршрутизации '$routeProvider' и фильтрация через '$routeParams', по условию 'courseId: params.courseId'.
На странице выбранного курса, выводятся: заголовок курса, одна картинка в <load-images></load-images>, и описание курса.


Задание желательно проверять на localhost (http://localhost/favoritApp), т.к. из-за особенностей подключения angular, могут возникнуть ошибки если просто открыть в браузере.
